# ⚡ DSA Tracker

A clean, dark-themed DSA (Data Structures & Algorithms) problem tracker built with pure HTML, CSS, and JavaScript — powered by Firebase for cross-device sync.

## ✨ Features

- 📂 Organize problems by patterns (Two Pointers, Sliding Window, etc.)
- ✅ Mark problems as solved and track progress per pattern
- 📊 Live stats — problems solved, current rank, remaining count
- 🔍 Search and filter by difficulty or platform
- ➕ Add custom patterns and problems
- ✏️ Rename or delete patterns and problems
- 🔗 Attach LeetCode, GFG, or YouTube links to each problem
- ☁️ Progress synced to cloud — access from any device
- 👤 Sign up / Sign in with Firebase Authentication
- 📱 Responsive design — works on mobile and desktop

## 🚀 Live Demo

[View Live on Vercel](https://dsavault.vercel.app)

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (Vanilla)
- Firebase Authentication
- Firebase Firestore (cloud database)
- Hosted on Vercel
